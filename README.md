# Airport-Optimization-Project
Gate Assignment & Tug Scheduling to Cut Pushback Delays

## Project Structure

- `airport_optimization.ipynb` - Main Jupyter notebook for Python analysis and optimization
- `data.csv` - Sample data file (replace with your actual data)

## Getting Started

1. Open `airport_optimization.ipynb` in Jupyter Notebook or JupyterLab
2. Replace `data.csv` with your actual data file
3. Run the notebook cells to perform analysis

## Data File Notes

The repository includes a sample `data.csv` file. For larger data files (up to 50MB):
- You can directly replace `data.csv` with your data file
- GitHub supports files up to 100MB in regular repositories
- For files larger than 50MB, consider using [Git LFS](https://git-lfs.github.com/)

## Requirements

- Python 3.x
- pandas
- numpy
- matplotlib
